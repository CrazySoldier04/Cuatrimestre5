Desarrollar un programa que pueda reutilizar las operaciones de POTENCIA Y RAIZ, ambos a la N.

La prueba se realiza en la clase PROGRAM.


Requisitos funcionales:
RF01.- El sistema debe de calcular la potencia de el número que el usuario ingrese a la n potencia
que el usuario establezca en un método programado para ello.
RF02.- El sistema debe de calcular la raíz a la n del número que el usuario ingrese en el método
programado para realizar el mismo.
RF03.- El número ingresado para calcular la potencia y la raíz debe de ser ENTERO y se validará que
cumpla con el formato requerido.

Requisito no funcional:
RNF01.- No se utilizará la libreria MATH.
RNF02.- La comprobación desplegará los resultados de una impresión de consola.
RNF03.- Si el número ingresado no cumple con el formato de tipo ENTERO, el sistema desplegará
un mensaje de error.
RNF04.- El sistema después de realizar el cálculo, desplegará un mensaje de despedida mediante
la consola.
