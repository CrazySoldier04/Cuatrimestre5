Desarrollar un programa para el cálculo del IVA en general en México y en la frontera. El
programa se emplea con métodos estáticos de una clase, dos métodos por cada funcionalidad.
Los datos de entrada serán en el método, la cantidad de la cuál se calcula dicho impuesto en
formato DOUBLE.

Requisitos funcionales:
RF01: El sistema debe de calcular el IVA tomando en consideración el IVA en la frontera y en el
resto del país. Se calcula éste último con la cifre del 16% de la cantidad que ingresa el usuario.
RF02: El IVA de la frontera lo calcula el sistema con base en 8% de la cantidad que solicita el
usuario.
RF03: El sistema permite validar los datos que ingresa el usuario para el calculo del IVA, siendo
el formato válido DOUBLE.

Requisitos no funcionales:
RNF01: El sistema comprobará la funcionalidad en el PROGRAM en el método main ingresando los datos
inline.
RNF02: El sistema después de mostrar los datos, lee una tecla del usuario para terminar por cerrarse.
