Desarrollar un programa que permita el envío de correo electrónico especificando destinatario,
asunto y msg.

Requisitos funcionales:
RF01: El sistema debe permitir mandar un mensaje de correo electrónico especificando asunto y
destinatario del mensaje.
RF02: El sistema validará que el asunto del mensaje no esté vacío.

Requisitos no funcionales:
RNF01: El sistema desplegará 3 textboxes en donde el usuario capturará el destinatario, el asunto y
el mensaje.
RNF02: Si el correo se envió, el sistema desplegará un mensaje de "correo enviado".
